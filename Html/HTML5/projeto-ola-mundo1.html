Tags de formatação especiais

Mark: Para marcar com cor Tag "mark". Para trocar a cor adicione a tag mark>style>background-color>cor desejada na frente da tag mark
Small: Texto pequeno Tag small no html5, as "letras miúdas do contrato" . Diminui o tamanho do texto.
Texto deletado Tag "del" para ser lido mas não considerado. Risca a parte citada.
Texto inserido Tag "ins" para indicar que o texto foi inserido depois. Marca a parte inserida posteriormente no texto.
Texto sobrescrito: Para escrever equações por exemplo x=225+6. Use a tag "sup" e coloque o símbolo dentro dela.
Texto subescrito: Para escrever coisas como h2o(sem a tag). H2O (com a tag "sub"). Coloque domente o símboo dentro da tag.

Imagens

Também gosto de Pókemon. Vou adicionar uma imagem com a tag "img". Use assim img src=(link da imagem dentro de aspas) alt=(descrição da foto detalhada para caso a imagem não seja exibida, ou para leitores e tela)

Citações simples

Citações simples sem quebra com a tag "q". Exemplo:
Como diria Gintama, um samurai de Edo: Doces são a melhor coisa do mundo!
Como vimos é só adicionar a tag "q" para fazer citações mais simples, sem links externos. Não alera nada no texto, mas dá créditos ao criador do conteúdo no código.
